## Getting Started

### What you'll need

* pnpm [PNPM installation](https://pnpm.io/installation).
* Node LTS >=18
* Docker

### Installation

To install the monorepo and all of its dependencies, run the following command at the root of the project:

```sh
pnpm install
```

## Start book_stock app

```sh
cd ./apps/books-stock
```

```sh
docker-compose up -d
```

```sh
pnpm run dev
```
